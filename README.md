[![Codecov](https://codecov.io/gh/hattya/md2html/branch/master/graph/badge.svg)](https://codecov.io/gh/hattya/md2html)


## Installation

```console
$ go install github.com/hattya/md2html@latest
```


## Usage

```console
$ md2html README.md README.html
```
